# 项目简介
* 支持模型切换。
* 支持全景图查看。
* 支持鼠标操控模型缩放旋转。
# 相关技术
* vue2主体框架。
* three.js生成场景、加载模型。
# 项目使用
* 安装依赖：npm install
* 项目运行：npm run serve
* 项目打包：npm run build
